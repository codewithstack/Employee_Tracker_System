# Suspicious Employee Detection System

## Overview
A comprehensive employee monitoring system that detects suspicious activities and generates real-time alerts to prevent data theft and insider threats.

## Features
- 👥 Employee Management
- 📝 Activity Tracking
- 🎯 Risk Score Calculation (0-100%)
- 🚨 Automatic Alert Generation
- 📊 Real-time Dashboard
- 🔍 6 Detection Algorithms:
  - Unusual Hours Detection
  - Data Export Detection
  - Multiple Location Detection
  - Failed Login Detection
  - Excessive Access Detection
  - Weekend Access Detection

## Technologies Used
- **Backend:** Python, Flask
- **Database:** SQLite
- **Frontend:** HTML, CSS, JavaScript
- **Libraries:** Flask-CORS, Flask-SQLAlchemy
